What is String Manipulation

String manipulation refers to the various operations and transformations that can be performed on strings, such as concatenation, slicing, replacing characters, converting cases, and more. It involves modifying or extracting specific parts of a string to achieve the desired outcome. Here's a basic program that demonstrates some common string manipulation operations in Python: 
# String Manipulation Example
text = "Hello, World!"

# Length of the string
length = len(text)
print("Length of the string:", length)

# Accessing individual characters
first_char = text[0]
last_char = text[-1]
print("First character:", first_char)
print("Last character:", last_char)

# Slicing the string
substring = text[7:12]
print("Substring:", substring)

# Concatenating strings
greeting = "Hello"
name = "John"
full_greeting = greeting + " " + name
print("Full greeting:", full_greeting)

# Converting case
uppercase = text.upper()
lowercase = text.lower()
print("Uppercase:", uppercase)
print("Lowercase:", lowercase)

# Replacing characters
new_text = text.replace("World", "Python")
print("Modified text:", new_text)

what is string replication operator (*)

The string replicator operator (*) in Python is used to repeat a string a certain number of times. It allows you to create a new string by concatenating multiple copies of the original string. Here's the definition of the string replicator operator and an example code that covers its usage: 
def string_replicator(string, n):
    """
    Function to replicate a string 'n' times using the string replicator operator (*).
    
    Args:
        string (str): The string to be replicated.
        n (int): The number of times the string should be repeated.
    
    Returns:
        str: The resulting string after replication.
    """
    return string * n

# Example usage
original_string = "Hello!"
replicated_string = string_replicator(original_string, 3)
print(replicated_string)

what are membership operators

Membership operators in Python are used to test whether a value or a variable is a member of a sequence (such as a string, list, or tuple) or a collection (such as a set or dictionary). The two membership operators available in Python are in and not in. Here's an example code that covers the topic of membership operators and includes their definitions: 
# Membership operators in Python

# Definition of membership operators
"""
The 'in' operator checks if a value is present in a sequence or collection.
It returns True if the value is found, and False otherwise.
Syntax: value in sequence/collection

The 'not in' operator checks if a value is not present in a sequence or collection.
It returns True if the value is not found, and False if it is found.
Syntax: value not in sequence/collection
"""

# Example usage of membership operators
# Lists
fruits = ['apple', 'banana', 'orange']

# 'in' operator
print('apple' in fruits)          # True
print('grape' in fruits)          # False

# 'not in' operator
print('apple' not in fruits)      # False
print('grape' not in fruits)      # True

# Strings
name = 'John Doe'

# 'in' operator
print('John' in name)             # True
print('Jane' in name)             # False

# 'not in' operator
print('John' not in name)         # False
print('Jane' not in name)         # True

# Sets
numbers = {1, 2, 3, 4, 5}

# 'in' operator
print(3 in numbers)               # True
print(6 in numbers)               # False

# 'not in' operator
print(3 not in numbers)           # False
print(6 not in numbers)           # True

# Dictionaries
person = {'name': 'John', 'age': 30, 'city': 'New York'}

# 'in' operator (checks keys)
print('name' in person)           # True
print('gender' in person)         # False

# 'not in' operator (checks keys)
print('name' not in person)       # False
print('gender' not in person)     # True

what are comparison operators

Certainly! Comparison operators in Python are used to compare values and determine the relationship between them. They return a Boolean value (True or False) based on the result of the comparison. Here's a code snippet that covers all the comparison operators in Python along with their definitions: 
# Comparison operators in Python

# Equal to (==) operator: Checks if two values are equal
x = 5
y = 10
print("Equal to (==) operator:", x == y)  # False

# Not equal to (!=) operator: Checks if two values are not equal
print("Not equal to (!=) operator:", x != y)  # True

# Greater than (>) operator: Checks if the left value is greater than the right value
print("Greater than (>) operator:", x > y)  # False

# Less than (<) operator: Checks if the left value is less than the right value
print("Less than (<) operator:", x < y)  # True

# Greater than or equal to (>=) operator: Checks if the left value is greater than or equal to the right value
print("Greater than or equal to (>=) operator:", x >= y)  # False

# Less than or equal to (<=) operator: Checks if the left value is less than or equal to the right value
print("Less than or equal to (<=) operator:", x <= y)  # True

what is string slicing

String slicing in Python refers to extracting a portion of a string by specifying the starting and ending indices. It allows you to retrieve a substring from a larger string based on its position. Here's an example code that covers various aspects of string slicing along with the definitions: 
# String slicing in Python

# Definition:
# String slicing is the process of extracting a portion of a string by specifying the starting and ending indices.

# Example 1: Basic slicing
string1 = "Hello, World!"
substring1 = string1[7:12]
print("Substring 1:", substring1)  # Output: "World"

# Example 2: Omitting start or end index
string2 = "Python Programming"
substring2_1 = string2[:6]
substring2_2 = string2[7:]
print("Substring 2.1:", substring2_1)  # Output: "Python"
print("Substring 2.2:", substring2_2)  # Output: "Programming"

# Example 3: Negative indices
string3 = "OpenAI"
substring3 = string3[-3:]
print("Substring 3:", substring3)  # Output: "AI"

# Example 4: Step value
string4 = "abcdefghijklmnopqrstuvwxyz"
substring4_1 = string4[::2]
substring4_2 = string4[1::3]
print("Substring 4.1:", substring4_1)  # Output: "acegikmoqsuwy"
print("Substring 4.2:", substring4_2)  # Output: "bfilorvx"

# Example 5: Reverse string
string5 = "Hello, World!"
substring5 = string5[::-1]
print("Substring 5:", substring5)  # Output: "!dlroW ,olleH"

What are string functions

In Python, string functions and methods are predefined operations that can be performed on strings. They provide a way to manipulate and work with strings in various ways. 
# String Functions and Methods

# len(): Returns the length of a string.
string = "Hello, World!"
length = len(string)
print(f"Length of the string: {length}")

# count(): Returns the number of occurrences of a substring in a string.
count = string.count("l")
print(f"Count of 'l' in the string: {count}")

# capitalize(): Converts the first character of a string to uppercase and the rest to lowercase.
capitalized_string = string.capitalize()
print(f"Capitalized string: {capitalized_string}")

# find(): Returns the index of the first occurrence of a substring in a string.
index = string.find("World")
print(f"Index of 'World' in the string: {index}")

# index(): Same as find(), but raises an exception if the substring is not found.
try:
    index = string.index("World")
    print(f"Index of 'World' in the string: {index}")
except ValueError:
    print("Substring not found!")

# isalnum(): Returns True if all characters in a string are alphanumeric (letters or digits).
is_alnum = string.isalnum()
print(f"Is the string alphanumeric? {is_alnum}")

# isalpha(): Returns True if all characters in a string are alphabetic (letters).
is_alpha = string.isalpha()
print(f"Is the string alphabetic? {is_alpha}")

# isdigit(): Returns True if all characters in a string are digits.
is_digit = string.isdigit()
print(f"Is the string numeric? {is_digit}")

# islower(): Returns True if all characters in a string are lowercase.
is_lower = string.islower()
print(f"Is the string lowercase? {is_lower}")

# isspace(): Returns True if all characters in a string are whitespace characters.
is_space = string.isspace()
print(f"Is the string whitespace? {is_space}")

# isupper(): Returns True if all characters in a string are uppercase.
is_upper = string.isupper()
print(f"Is the string uppercase? {is_upper}")

# lower(): Converts all characters in a string to lowercase.
lower_string = string.lower()
print(f"Lowercase string: {lower_string}")

# upper(): Converts all characters in a string to uppercase.
upper_string = string.upper()
print(f"Uppercase string: {upper_string}")

# lstrip(): Removes leading whitespace characters from a string.
lstripped_string = string.lstrip()
print(f"Left-stripped string: {lstripped_string}")

# rstrip(): Removes trailing whitespace characters from a string.
rstripped_string = string.rstrip()
print(f"Right-stripped string: {rstripped_string}")

# strip(): Removes leading and trailing whitespace characters from a string.
stripped_string = string.strip()
print(f"Stripped string: {stripped_string}")

# istitle(): Returns True if a string is titlecased (each word starts with an uppercase character and the remaining characters are lowercase).
is_title = string.istitle()
print(f"Is the string titlecased? {is_title}")

# title(): Converts a string to titlecase (each word starts with an uppercase character and the remaining characters are lowercase).
title_string = string.title()
print(f"Titlecased string: {title_string}")

# startswith(): Returns True if a string starts with a specified substring.
starts_with = string.startswith("Hello")
print(f"Does the string start with 'Hello'? {starts_with}")

# endswith(): Returns True if a string ends with a specified substring.
ends_with = string.endswith("World!")
print(f"Does the string end with 'World!'? {ends_with}")

# replace(): Replaces all occurrences of a substring in a string with another substring.
replaced_string = string.replace("Hello", "Hi")
print(f"Replaced string: {replaced_string}")

# join(): Joins elements of an iterable (e.g., list) into a single string using a specified separator.
words = ["Hello", "World!"]
joined_string = " ".join(words)
print(f"Joined string: {joined_string}")

# split(): Splits a string into a list of substrings based on a specified separator.
split_string = string.split(",")
print(f"Split string: {split_string}")

# partition(): Splits a string into a tuple containing the substring before the first occurrence of a separator, the separator itself, and the substring after it.
partitioned_string = string.partition(",")
print(f"Partitioned string: {partitioned_string}")

import re

# Function to validate the email format
def is_valid_email(email):
    pattern = r'^[\w\.-]+@[\w\.-]+\.\w+$'
    return re.match(pattern, email)

# Function to validate the phone number format
def is_valid_phone_number(phone_number):
    pattern = r'^\d{10}$'
    return re.match(pattern, phone_number)

# Taking input from the user
username = input("Enter your username: ")
phone_number = input("Enter your phone number: ")
email = input("Enter your email: ")

# Validating the email and phone number
if is_valid_email(email):
    print("Email is valid.")
else:
    print("Email is not valid.")

if is_valid_phone_number(phone_number):
    print("Phone number is valid.")
else:
    print("Phone number is not valid.")
