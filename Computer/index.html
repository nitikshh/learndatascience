# Count the number of occurrences of each keyword in a given code snippet
def count_keywords(code):
    keyword_count = {}
    
    # Initialize the count for each keyword to 0
    for keyword in keywords:
        keyword_count[keyword] = 0
    
    # Split the code into individual words
    words = code.split()
    
    # Iterate over each word in the code
    for word in words:
        # Check if the word is a keyword
        if word in keywords:
            # Increment the count for the keyword
            keyword_count[word] += 1
    
    return keyword_count

# Example usage of the count_keywords function
code_snippet = '''
def calculate_sum(numbers):
    total = 0
    for num in numbers:
        total += num
    return total
'''

keyword_occurrences = count_keywords(code_snippet)

# Print the count of each keyword
for keyword, count in keyword_occurrences.items():
    print(f"{keyword}: {count}")
This code demonstrates how to count the occurrences of Python keywords in a given code snippet. The `count_keywords` function takes a code snippet as input and returns a dictionary containing the count of each keyword. It initializes the count for each keyword to 0, splits the code into individual words, and then iterates over each word to check if it matches any of the keywords. If a match is found, the count for that keyword is incremented.
In the example usage, a code snippet is provided, and the `count_keywords` function is called on it. The resulting keyword occurrences are then printed using a loop.
Note that this code assumes the code snippet is given as a string and separated by whitespace. It may not accurately handle code with complex syntax or comments.

# Function to check if a number is even or odd
def check_even_odd(number):
    if number % 2 == 0:
        return "Even"
    else:
        return "Odd"

# Function to check if a string is a palindrome
def check_palindrome(string):
    reversed_string = string[::-1]
    if string == reversed_string:
        return "Palindrome"
    else:
        return "Not a Palindrome"

# Function to check if a number is prime
def check_prime(number):
    if number <= 1:
        return "Not Prime"
    for i in range(2, int(number ** 0.5) + 1):
        if number % i == 0:
            return "Not Prime"
    return "Prime"

# Test the functions

# Function to calculate the sum of two numbers
def add_numbers(a, b):
    return a + b

# Function to calculate the difference between two numbers
def subtract_numbers(a, b):
    return a - b

# Function to calculate the product of two numbers
def multiply_numbers(a, b):
    return a * b

# Function to calculate the division of two numbers
def divide_numbers(a, b):
    if b == 0:
        return "Error: Cannot divide by zero"
    return a / b

# Main program
num1 = 10
num2 = 5

# Call the add_numbers function and print the result
sum_result = add_numbers(num1, num2)

# Call the subtract_numbers function and print the result

print("Difference:", diff_result)

# Call the multiply_numbers function and print the result

print("Product:", prod_result)

# Call the divide_numbers function and print the result
div_result = divide_numbers(num1, num2)
print("Division:", div_result)
In this code, we define four functions: `add_numbers`, `subtract_numbers`, `multiply_numbers`, and `divide_numbers`. These functions perform basic arithmetic operations on two numbers. The main program demonstrates how to call these functions with example numbers and print the results.

How many ways are there in Python to represent an integer literal ?

Solution. Python allows tree types of integer literals:
(a) Decimal Integer Literals. An integer literal consisting of a sequence of digits is taken to b. decimal integer literal unless it begins with 0 (digit zero).
(b) Octal Integer Literals. A sequence of digits starting with Oo (digit zero followed by letter is taken to be an octal integer.
