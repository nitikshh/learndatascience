what are dictionaries

Dictionaries in Python are unordered collections of key-value pairs. They are used to store and retrieve data in an efficient manner. Keys in a dictionary are unique and immutable, while values can be of any data type and can be duplicated.
Here's an example code that covers the definition and demonstrates the various operations related to dictionaries in Python: 
# Define a dictionary
my_dict = {
    'name': 'John',
    'age': 25,
    'city': 'New York'
}

# Accessing values using keys
print("Name:", my_dict['name'])
print("Age:", my_dict['age'])
print("City:", my_dict['city'])

# Modifying values
my_dict['age'] = 26
print("Updated age:", my_dict['age'])

# Adding a new key-value pair
my_dict['occupation'] = 'Engineer'
print("Occupation:", my_dict['occupation'])

# Removing a key-value pair
del my_dict['city']
print("After deleting 'city' key:", my_dict)

# Checking if a key exists
if 'name' in my_dict:
    print("Key 'name' exists in the dictionary")

# Getting all keys and values
print("Keys:", my_dict.keys())
print("Values:", my_dict.values())

# Iterating over keys and values
for key, value in my_dict.items():
    print(key, ":", value)

# Clearing the dictionary
my_dict.clear()
print("Dictionary after clearing:", my_dict)

Dictionaries Functions and Methods

Here's an example code that covers the functions and methods of dictionaries in Python along with their definitions: 
# Dictionary Functions and Methods

# 1. len() - Returns the number of key-value pairs in the dictionary.
dictionary = {'apple': 3, 'banana': 5, 'orange': 2}
length = len(dictionary)
print("len(dictionary) returns:", length)
# Output: len(dictionary) returns: 3


# 2. dict() - Creates a new dictionary.
new_dictionary = dict()
print("dict() creates an empty dictionary:", new_dictionary)
# Output: dict() creates an empty dictionary: {}


# 3. clear() - Removes all items from the dictionary.
dictionary.clear()
print("dictionary.clear() clears the dictionary:", dictionary)
# Output: dictionary.clear() clears the dictionary: {}


# 4. copy() - Returns a shallow copy of the dictionary.
copy_dictionary = new_dictionary.copy()
print("copy_dictionary = new_dictionary.copy() creates a shallow copy:", copy_dictionary)
# Output: copy_dictionary = new_dictionary.copy() creates a shallow copy: {}


# Dictionary Methods:

# 1. keys() - Returns a view object that contains all the keys in the dictionary.
keys = copy_dictionary.keys()
print("copy_dictionary.keys() returns:", keys)
# Output: copy_dictionary.keys() returns: dict_keys([])


# 2. values() - Returns a view object that contains all the values in the dictionary.
values = copy_dictionary.values()
print("copy_dictionary.values() returns:", values)
# Output: copy_dictionary.values() returns: dict_values([])


# 3. items() - Returns a view object that contains all the key-value pairs in the dictionary as tuples.
items = copy_dictionary.items()
print("copy_dictionary.items() returns:", items)
# Output: copy_dictionary.items() returns: dict_items([])


# 4. get(key[, default]) - Returns the value for the given key if it exists, otherwise returns the default value.
value = copy_dictionary.get('key', 'default_value')
print("copy_dictionary.get('key', 'default_value') returns:", value)
# Output: copy_dictionary.get('key', 'default_value') returns: default_value


# 5. pop(key[, default]) - Removes the item with the specified key and returns its value. If the key is not found, returns the default value.
value = copy_dictionary.pop('key', 'default_value')
print("copy_dictionary.pop('key', 'default_value') returns:", value)
# Output: copy_dictionary.pop('key', 'default_value') returns: default_value


# 6. update(iterable) - Updates the dictionary with the key-value pairs from the specified iterable.
copy_dictionary.update({'key': 'value'})
print("copy_dictionary.update({'key': 'value'}) updates the dictionary:", copy_dictionary)
# Output: copy_dictionary.update({'key': 'value'}) updates the dictionary: {'key': 'value'}

Creating Dictionaries from Keys

Creating dictionaries from keys in Python refers to the process of generating a dictionary where the keys are provided, and the corresponding values are assigned either a default value or are derived from some calculation or logic. Below is an example code that covers this topic: 
# Define the keys
keys = ['apple', 'banana', 'cherry']

# Create a dictionary with default values
dictionary_with_default = {key: 0 for key in keys}
print("Dictionary with default values:", dictionary_with_default)

# Create a dictionary with derived values
dictionary_with_derived = {key: len(key) for key in keys}
print("Dictionary with derived values:", dictionary_with_derived)

# Create a dictionary with user input values
dictionary_with_user_input = {key: input(f"Enter a value for {key}: ") for key in keys}
print("Dictionary with user input values:", dictionary_with_user_input)
