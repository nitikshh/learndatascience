Finding information by describe() 

df.describe()

Explanation : The code `df.describe()` is a common method used in data science to generate descriptive statistics of a DataFrame or a dataset. Let's break down what each part of the code does:

- `df` refers to a DataFrame object, which is a two-dimensional tabular data structure commonly used in data analysis. It can be created from various data sources such as CSV files, SQL databases, or Excel sheets.

- The `describe()` method is called on the DataFrame object `df`. This method calculates various summary statistics of the numerical columns in the DataFrame, such as count, mean, standard deviation, minimum, quartiles, and maximum values. By default, it excludes any non-numeric columns from the analysis.

When `df.describe()` is executed, it returns a new DataFrame that contains the calculated summary statistics. The resulting DataFrame has the following columns:

- `count`: The number of non-null values in each column. This represents the number of observations available for each numerical feature.

- `mean`: The average value of each column. It is calculated by summing up all the values in the column and dividing by the number of non-null values.

- `std`: The standard deviation, which measures the spread or dispersion of the values in each column. It quantifies the amount of variation or deviation from the mean.

- `min`: The minimum value observed in each column. It represents the smallest value present.

- `25%`, `50%`, and `75%`: These are the quartiles of the data distribution. The 25% quartile (also known as the first quartile or lower quartile) represents the value below which 25% of the data falls. The 50% quartile (also known as the median) represents the value below which 50% of the data falls. The 75% quartile (also known as the third quartile or upper quartile) represents the value below which 75% of the data falls.

- `max`: The maximum value observed in each column. It represents the largest value present.

The `describe()` method provides a quick way to obtain an overview of the distribution and basic statistics of the numerical data in a DataFrame. It helps in identifying potential outliers, understanding the central tendency and variability of the data, and gaining initial insights into the dataset.

Here's a breakdown of the information provided by `df.info()`:

2. Non-Null Count: For each column, `df.info()` shows the number of non-null values. This information helps in identifying missing or incomplete data. By knowing the number of non-null values, you can assess the quality and completeness of your dataset.

3. Data Types: The method also displays the data type of each column. Common data types include integers, floats, strings, dates, and categorical variables. Knowing the data types is important for performing appropriate data manipulation and analysis. For example, numerical operations can only be performed on numeric data types.

The output of `df.info()` is usually presented in a tabular format, with columns for column names, non-null counts, and data types. It provides a concise summary of the DataFrame, giving you a quick overview of the data structure, missing values, and data types of the columns. This summary is helpful for initial data exploration and understanding the characteristics of the dataset.

Some other Graphs 

import matplotlib.pyplot as plt
import seaborn as sns

# Assuming you have the Titanic dataset stored in a DataFrame called 'df'

# Bar Plot: Survival Count by Sex
sns.set_style("whitegrid")
plt.figure(figsize=(8, 6))
sns.countplot(data=df, x='Sex', hue='Survived')
plt.title("Survival Count by Sex")
plt.xlabel("Sex")
plt.ylabel("Count")
plt.show()

# Histogram: Age Distribution
plt.figure(figsize=(8, 6))
sns.histplot(data=df, x='Age', kde=True)
plt.title("Age Distribution")
plt.xlabel("Age")
plt.ylabel("Count")
plt.show()

# Scatter Plot: Fare vs. Age
plt.figure(figsize=(8, 6))
sns.scatterplot(data=df, x='Age', y='Fare', hue='Survived')
plt.title("Fare vs. Age")
plt.xlabel("Age")
plt.ylabel("Fare")
plt.show()

Explanation : The first plot is a bar plot that shows the count of survivors and non-survivors based on gender. It uses the countplot function from Seaborn to create a bar chart with the 'Sex' column on the x-axis and the count of occurrences on the y-axis. The 'Survived' column is used to differentiate the bars by hue.

The second plot is a histogram that displays the distribution of ages among the passengers. It uses the histplot function from Seaborn to create a histogram with the 'Age' column on the x-axis and the count of occurrences on the y-axis. The kde=True argument adds a kernel density estimation line to the plot.

The third plot is a scatter plot that shows the relationship between the fare paid and the age of the passengers. It uses the scatterplot function from Seaborn to create a scatter plot with the 'Age' column on the x-axis and the 'Fare' column on the y-axis. The 'Survived' column is used to differentiate the points by hue.
